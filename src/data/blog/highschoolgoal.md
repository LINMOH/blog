目前来看完成度还算不错的，当年定的目标不算太高。

现在是时候着手一下高中三年的计划了。

首先，上了高中，首先面对的就是高考。对于大学的目标，我分为两个维度：国内本和美本。国内本不必多说，就是传统高考路线，对我是没有太大优势的，并且我在山东有点苦。国内的目标，从低到高分别是：杭电、华东理工、山大、浙大。走国内本的路线的话，读研我觉得一定要出去读的，这就是后话了，估计会在 29 年的“高中总结 & 大学计划”里提到（笑）。

至于美本，我觉得值得冲一下，毕竟国内的本科真有点垃圾。美本的话，我觉得会去 Harvard/MIT/Princeton 等有 Need-Blind 的大学为奋斗目标。但是学费是个问题，我们家也只能算小康吧，不过我觉得还是值得一试的。如果未来能留在海外发展，那自然很好；如果不能，国内也并非没有机会。

## 学期任务

1. 高一上：打牢基础

- 问题：词汇量断层、校内压轴题不稳
- 文化课：死磕数学与化学。校内数学目标单科校前5，学必修二，目标是领先学校进度一个单元。化学必须回归校前10。GPA 稳定在校前 2%。
- 英语：背完高考3500词，无缝衔接托福核心词。放弃盲目磨耳朵，精听 TED-Ed / NPR 科学类短篇 / CS 英文播客。
- 计算机：主攻 OSTEP 的 CPU 虚拟化与内存管理部分。
- 竞赛：CSP-S 冲刺二等。
- AP：启动微积分 BC，物理 C，CSA。

2. 高一下：大量输入

- 文化课：GPA 稳定在校前 2%。
- 英语：词汇量冲7000，托福首战目标 80+。
- 开源：将精力放在为 TATEN 自己的开源项目（TATEN-OJ、AniDay）编写高质量英文文档、核心在 Sandboxing 上。
- 数学：校内拿下圆锥曲线。为高二的离散和线代打下基础。
- AP：考微积分 BC 和物理 C 和 CSA；

3. 高二上：国际化与提分

- 文化课：GPA 稳定在校前 2%。
- 英语：托福二战，目标 98+。阅读和听力应该接近满分。
- 数学：校内攻克选修二三。
- 计算机：精读 OSTEP 的并发与持久化。做 MIT6.S081 的 Lab，重点攻克 Util, Syscall, Page tables, Copy-on-Write, Multithreading，其余视精力而定。
- 计算机：将 TATEN 社区全网推广，完善英文 Readme。尝试向 Linux Kernel 或是主流开源基础设施提交实质性 PR。
- 竞赛：CSP-S / NOIP 冲刺一等奖。若拿到，美本/国内双线全部受益；若二等，立即停止所有 OI 训练。

4. 高二下：标化收尾与高等数学

- 文化课：GPA 稳定在校前 2%。
- 全力备战 SAT。数学必须 800 满分，阅读目标 700+。
- AP：再战微积分 BC 和物理 C（视情况）
